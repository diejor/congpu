#define ERR(cond, fmt, ...)

ERR:

If the condition is true, logs an error message at LL_ERROR level (including the stringized condition) and returns from the function.

Usage: ERR(val < 0, "Invalid negative value: %d", val);

#define WARN_COND(cond, fmt, ...)

WARN_COND:

If the condition is true, logs a warning message at LL_WARN level (including the stringized condition) but does not return.
